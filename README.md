# Databricks Practice

A hands-on repository for building data pipelines in Databricks and organizing study materials for Databricks certification exams.

## Purpose

- **Pipeline Development** - Practice building ETL/ELT pipelines using PySpark, Delta Lake, and Delta Live Tables
- **Exam Prep** - Store and organize study notes, practice questions, and reference materials for Databricks certification

## Tech Stack

- **Databricks** - Unified analytics platform
- **PySpark** - Distributed data processing
- **Delta Lake** - ACID transactions on data lakes
- **SQL** - Databricks SQL analytics

## Getting Started

1. Clone this repo
2. Import notebooks into your Databricks workspace
3. Connect to your cluster and run
